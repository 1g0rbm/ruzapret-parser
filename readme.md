### Ruzapret парсер
Утилита парсит открытыю информацию о заблокированных сайтах и сохраняет в файл

### Пререквизиты
nodejs >= 14

### Установка
```bash
git clone git@github.com:1g0rbm/ruzapret-parser.git
cd ruzapret-parser
npm install
npm link
```

### Использование
```bash
load-banned-site /path/to/dir
```
